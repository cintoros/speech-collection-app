Welcome to the Speech Collection App.
This web application was built for the purpose of collecting Swiss German speech with the corresponding Standard German transcript to train Swiss German speech-to-text models.
You can help us build a freely available Swiss German speech-to-text model by providing speech (Record function) or by doublechecking the recordings of other users (Check function).
Any contribution is much appreciated!

Record

Record Swiss German speech for a Standard German sentence. Please think about the Swiss German translation of the sentence first before starting to record.
When you have spoken a sentence, you can play it back (simply with the play symbol).
If you are not satisfied, you can simply repeat the recording (press the start button again).
If you are satisfied, press the blue submit button.
  • Start: Start recording speech
  • Skip: I don't want to read this sentence aloud, give me another one.
  • Not a sentence: Transcript is not Standard German or is not a single sentence or phrase. In a pre-processing step, longer texts are automatically split to sentences, so use this button also if you suspect that the sentence splitting went wrong.
  • Sensitive data: Transcript and / or recording contains sensitive data and should not be published.
Start Recording

Check

Doublecheck whether a transcript is a Standard German sentence and the corresponding recording is a correct Swiss German translation.
  • Correct: Transcript is a Standard German sentence and recording is a correct Swiss German translation
  • Wrong: Recording is not a correct Swiss German translation
  • Skip: Unsure whether recording is a correct Swiss German translation or not
  • Not a sentence: Transcript is not Standard German or is not a single sentence or phrase. In a pre-processing step, longer texts are automatically split to sentences, so use this button also if you suspect that the sentence splitting went wrong.
  • Sensitive data: Transcript and / or recording contains sensitive data and should not be published.
